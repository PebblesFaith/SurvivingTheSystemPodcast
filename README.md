# Surviving the System — Hosted by Sarai Ajai

This repository contains the source files for the official website of **Surviving the System**, a live investigative podcast documenting real experiences, systemic failures, and evidence-based analysis.

The website is published through **GitHub Pages** and connected to  
**https://SurvivingTheSystemPodcast.com**.

### Features
- Official podcast landing page
- Host introduction and mission
- Episode links (YouTube & social platforms)
- Contact and updates

### Tech Stack
- HTML, CSS, JS
- GitHub Pages (static hosting)
- Custom domain via IONOS
